题目:
```
    设某棵二叉树中有2000个结点，则该二叉树的最小高度为——————
```
二叉树中有2000个结点，如果想要最小高度，那么就是需要满二叉树了。满二叉树高度与结点个数的关系是

```math
num=2^n
```
2000>2^10

2000<2^11

最小高度为11层
